Decisions

DecisionReason
Name the project ClipToAction.The name matches the workflow: clip in, action out.
Only the video link should be required.The user will not add title, notes, category, or action while sharing from mobile.
Use a standalone HTML file first.Fast to use, no login, no install, low token/time cost.
Keep AI automation as phase 2.The workflow should be useful even before transcript automation exists.
